# django_todo_app_api

This project is a Simple ToDo List App using Django and API using Django Rest Framework

ToDo List App cointain following features

1) Login
2) Register User
3) ToDo List
4) ToDo Create
5) ToDo Update
6) ToDo Delete
7) ToDo Search

ToDo List API cointain following features

1) Basic Authentication and Session Authentication
3) ToDo List
4) ToDo Create
5) ToDo Update
6) ToDo Delete
7) ToDo Search

How to Create Project can create using requirements.txt file

1) Setup Virtual environment using pipenv 
2) Create Django project config
3) Create Django App todos
4) In a normal Django app we need to add urls, models(Tag with many to many relationship and Todo), views, and templates
5) Make migrations
6) Create superuser
7) Install Django Rest Framework using pipenv
8) Create Django App api
9) Create a serializers and urls file to access api

Can Access project URLs using

Django Admin Dashboard : http://127.0.0.1:8000/admin/

Django ToDo App Login : http://127.0.0.1:8000/

Django ToDo API : http://127.0.0.1:8000/apis/v1/
